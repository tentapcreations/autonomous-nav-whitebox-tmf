# 🚀 Autonomous-Nav-Whitebox (TMF)

### A Pluggable, Lightweight, and White-Box Architecture for Autonomous Navigation & Traversability Estimation

---

## 💡 Overview
**TMF (Traversable Medium Framework)** is a transparent safety and navigation engine designed for autonomous driving and robotics. It bridges the gap between complex perception and explainable control, transforming black-box systems into deterministic, white-box physical logic.

Evolving from a passive shadow observer into a multi-form safety envelope, TMF now supports three flexible deployment modes to fit any architecture:
1. **Direct Insertion (直接插入):** Seamlessly plugs into existing navigation stacks (e.g., ROS/Nav2) with high compatibility.
2. **Standalone Core (新系统重构):** Serves as the lightweight, native foundation for next-generation navigation systems.
3. **Bypass Auditor (旁路审计):** Operates as a zero-invasive, passive binary mask architecture for real-time perception auditing.

## 🌟 Why TMF? (核心优势)
* ⚪ **White-Box & Explainable (黑盒变白盒):** Provides fully traceable physical/semantic logic to ensure safety-critical decision-making.
* ⚡ **Lightweight & Efficient (节约算力/简化架构):** Drastically reduces computational overhead, making it ideal for resource-constrained edge platforms.
* 🔌 **Pluggable & Versatile (即插即用):** Zero-invasive to core code; adapts effortlessly from high-end autonomous vehicles to embedded robotic systems.
